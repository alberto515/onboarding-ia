# Onboarding IA

Proyecto de onboarding que incluye un script básico en Python para verificar el entorno y dar la bienvenida.

## Descripción

**onboarding-ia** es un repositorio de arranque con un programa sencillo que imprime un saludo y la fecha actual en el terminal. Sirve como punto de partida para configurar el entorno de desarrollo y familiarizarse con el flujo del proyecto.

## Contenido

- **`hola_mundo.py`**: Script que muestra por consola el mensaje "Hola Mundo" y la fecha del día en formato `DD-MM-YYYY`.

## Requisitos

- **Python 3.x** (solo se usa la biblioteca estándar; no hay dependencias externas).

## Cómo ejecutar

Desde la raíz del proyecto:

```bash
python hola_mundo.py
```

Salida esperada (ejemplo):

```
Hola Mundo
10-02-2025
```
